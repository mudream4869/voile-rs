# Voile

A personal book reading app.

## Target

### Will Provide

* Reading simple book form. For example:
    * A folder of images. (Manga?)
    * A folder of txts. (Novel?)
    * One txt. (Novel?)
    * pdf
    * epub
* Upload/taging/editing info of books.
* Single binary.

### Won't Provide

* Multi-user
* Crawler

## How to run

### Build frontend

```bash
cd frontend
yarn build
```

### Start

```bash
cargo run configs/voile.toml
```
